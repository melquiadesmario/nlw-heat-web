## Stage 2 – Aula prática

Nessa aula criaremos o front-end web da nossa aplicação utilizando ReactJS. Além disso, criaremos nosso projeto utilizando o Vite, uma ferramenta extremamente performática, TypeScript e CSS modules. Vamos construir toda interface da aplicação do zero com Flexbox e Grid System, além de criar animações utilizando Framer Motion.
